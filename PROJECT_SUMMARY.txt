URL PARAMETERS
================================================================================

Configure the app via URL parameters:

/?org_ids=516_512_513          - Specific organizations
  &facility_ids=101_102         - Filter by facilities
  &program_types=class,clinic   - Filter by type
  &sports=yoga,tennis           - Filter by activity
  &program_name=yoga            - Search by name
  &start_date=2025-01-20        - Filter start date
  &end_date=2025-02-20          - Filter end date
  &show_filters=facility,date   - Show only these filters
  &view_mode=discovery          - View mode (discovery/schedule/direct)

Examples:
  http://localhost:5173/?sports=yoga
  http://localhost:5173/?org_ids=516&show_filters=program_type,date_range
  http://localhost:5173/?program_types=class&program_name=beginner
