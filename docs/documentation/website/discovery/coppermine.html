Adding a program page to gocoppermine.com

How to add a new sport page (lacrosse, dance, …) that shows live Bond programs and takes registrations — about 5 minutes per page in WordPress.

Already set up!

Bond scriptInstalled site-wide via the WPCode plugin ("Bond Discovery host kit" snippet). Powers every Bond page.
All-programs page/discovery/programs — shows every program. In WordPress it's the page titled "Programs (Bond)".
Soccer page/discovery/programs/soccer — the first sport page. Duplicate this one to add each new sport (steps below).
Register page/discovery/register — checkout opens here in a new tab. One register page serves all program pages; never create a second one.
Search enginesAll /discovery/… pages are hidden from Google for now (a WPCode snippet). Bond will remove this at launch.

Add a new sport page

Before you start: ask Bond (charlie.lynch@bondsports.co) to create the sport's Bond page. Bond replies with the sport's embed slug — e.g. coppermine-lacrosse. You need it for step 4.

The easiest way is to duplicate the existing Soccer page — the copy inherits the right parent, URL structure, and embed code automatically:

1

In wp-admin: Pages → find Soccer (under Programs (Bond)) → hover it → click New Draft. That makes a copy and opens it for editing.

2

Change the title from "Soccer" to the new sport, e.g. Lacrosse.

3

Click the Slug row in the right-hand panel and change it to the sport name, e.g. lacrosse (the copy starts as something like soccer-2). Confirm the full URL reads gocoppermine.com/discovery/programs/lacrosse/.

4

Click the Custom HTML block in the page body and change one thing: the data-bond-slug value — replace it with the slug Bond sent you (e.g. coppermine-lacrosse). Everything else stays as-is.

5

Publish the page.

6

In the black toolbar at the top: Delete Cache → Clear All Cache (WP Fastest Cache). Required after every change.

7

Open the page on the live site. It should show only that sport's programs, and clicking Register should open checkout in a new tab.

Building a page from scratch instead? Create a page with Parent → "Programs (Bond)" (the one with "(Bond)" in the name, not the old Programs page), then add a Custom HTML block with this snippet — change only the slug:

<div
  data-bond-host
  data-bond-slug="coppermine-lacrosse"
  data-bond-discovery-base="https://discovery.bondsports.co"
></div>

If something looks wrong

Old or missing contentClear the cache (step 6), then hard-refresh the browser (Cmd/Ctrl + Shift + R).
Embed area is emptyMake sure the page is published (editor previews don't run scripts). If it's empty only before accepting the cookie banner, tell Bond — the consent tool may be blocking the script.
Register lands on the homepagePath mismatch between the site and Bond's settings — contact Bond, don't change pages.
Wrong programs showingThe data-bond-slug in the Custom HTML block doesn't match the sport — check step 4.
